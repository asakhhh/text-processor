# Text processor (Go)

`go run . <input file> <output file>`

Processes tokens in the input file and writes updated text in the output file.

- (hex) - converts previous hexadecimal number into equivalent decimal number
- (bin) - converts previous binary number into equivalent decimal number
- (up) - turns previous word to uppercase
- (low) - turns previous word to lowercase
- (cap) - capitalizes previous word
- (up/low/cap, N) - applies specified transformation to N previous words

# Текстовый процессор (Go)

`go run . <input file> <output file>`

Обрабатывает токены указанные во входном файле и выводит результат в выходной файл.

- (hex) - Заменяет предыдущее слово, считая его шестнадцатеричным числом, на десятичное
- (bin) - Заменяет предыдущее слово, считая его двоичным числом, на десятичное
- (up) - Преобразует предыдущее слово в ВЕРХНИЙ РЕГИСТР
- (low) - Преобразует предыдущее слово в нижний регистр
- (cap) - Преобразует предыдущее слово так, чтобы первая буква была заглавной
- (up/low/cap, N) - Применяет соответствующее преобразование к N предыдущим словам
